# BUPT-Resource
此项目主要通过收集北邮各院的复习资料，PPT，考试题目等信息

## 如何贡献资源
贡献资源主要通过PR(Pull Request)，进行提交

贡献时需要标明

1. 提交类型（考试/复习资料/课件/答案）
2. 来源（上课群/学长/网上资料）

## 如何下载

1. 下载时可以通过github的download按钮进行下载，可能速度较慢，需要使用梯子
2. 下载也可以使用`git clone` 下载，如果网速较慢可以通过`git clone --depth=1` 进行下载。下载时尽量开启全局代理
